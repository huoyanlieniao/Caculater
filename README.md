#逻辑规则

##输入判断规则

- [ ] +-*/. 除了和()连接其他不能连接

- [ ] ()不能连在一起
- [ ] 除法分母不能为0





##其他规则

###首位规则

- [ ] 首位设置为空
- [ ] .*/+不能为首位
- [ ] -为首位时输入数字为负数

###尾位规则

- [ ] 尾位不能为符号

###显示规则

- [ ] 可以显示历史计算，清除后清除显示
- [ ] =计算后显示=和结果
